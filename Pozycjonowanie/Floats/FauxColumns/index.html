Faux Columns for Liquid Layouts

Using CSS for layouts can be a problem when using backgrounds for two columns that are not of equal length.

If your two columns each have a different background color, how do you make the colours extend to the bottom of the page using css?

The best solution involves tiling a background on the page, to give the appearance of two distinct columns that extend the full length of the page.

This solution can be used to apply a background to a page when the layout is liquid, using background positioning.

Background positioning can allow us to give our background the appearance of being liquid. When you positioning a background using pixels, you position the top left corner of the image.

When you position a background using percentages or keywords, you position a point within the image itself.
